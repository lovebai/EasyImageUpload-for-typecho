# EasyImageUpload-for-typecho
Typecho 博客程序上传到简单图床插件，基于兰空图床typecho插件二次开发，由于简单图床提供的api接口不够丰富，插件使用可能会有问题

# 使用方法
 - 登录图床后台->图床安全->高级设置->开启API上传
 - 后台API设置中获取网站生成的token
 - 上传插件到 `usr/plugins` 下，到typecho后台启用插件即可
